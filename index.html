Things I've built.

Smart Parking System

An automated real-time parking management system that detects vehicle occupancy using IR sensors and displays live slot availability — no app, no internet, just hardware that just works.

  • IR sensors wired to each parking slot to detect vehicle presence with <50ms response time
  • Arduino Uno processes sensor arrays and drives a display showing available vs. occupied slots
  • Implemented queue and struct data structures in C for efficient slot state tracking
  • Reduces parking search time by giving drivers immediate, accurate slot availability info
Arduino Uno IR Sensors C / C++ Embedded Systems Data Structures

Vision-Controlled Robotic Arm

A browser-based gesture recognition system that classifies webcam hand poses with a TensorFlow.js model and sends servo commands over Web Serial to an Arduino Leonardo — controlling a real 4-DOF robotic arm in real time.

  • Trained custom image classifier on Google Teachable Machine; exported to TensorFlow.js for in-browser inference
  • Built HTML/JS controller using Web Serial API for live bidirectional communication with Arduino Leonardo
  • Mapped 4 gesture classes to claw, elbow, shoulder, and base servo movements with smooth PWM control
  • Achieved near-zero-latency control loop entirely within the browser — no backend server required
Google Teachable Machine TensorFlow.js Arduino Leonardo Web Serial API JavaScript Computer Vision
